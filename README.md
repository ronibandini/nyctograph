# nyctograph
Nyctograph machine based in Lewis Carroll invention

# Connections

LCD 16 x2 SDA to A4, SCL to A5, VCC and GND
Printer Yellow cable to D12 and Green to D11
Rotary Encoder SW to D5, Signals to D6, D7
Led + to D8 and GND
